## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Balaji-webdev/fitPeo-dashboard
```

2. Navigate to project directory
```bash
cd Dashboard
```

3. Install dependencies
```bash
npm install
```

4. Start development server
```bash
npm run dev
```
